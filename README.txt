Projectile Motion Simulation 

Description:

  This project is a Python-based simulation of projectile motion. 
  It allows the user to enter initial values and visually see how an object moves when thrown at an angle.

Features:

User input for:
  Speed (m/s)
  Angle (degrees)
  Gravity (m/s²)

Real-time animation of projectile motion

Displays:

  Range (distance)
  Maximum height
  Time of flight


Technologies Used:

  Python
  Pygame
  pygame_gui


How to Run:

 1.Install required libraries:

   pip install pygame pygame_gui
   
 2.Run the program:

   python main.py
   

How to Use:

 1. Enter values in the input boxes
 2. Click **START**
 3. Observe the motion and results on screen

Output:

The program calculates and displays:

  Total horizontal distance (Range)
  Maximum height reached
  Total time the object stays in air
